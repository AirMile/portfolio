# Smart Suggestions Reference

## React Projects

### Essential Suggestions
- **TypeScript**: Type safety, better IDE support, easier refactoring
- **React Router**: Multi-page navigation (react-router-dom)
- **CSS Solution**: Tailwind CSS, styled-components, or CSS Modules

### State Management Options
- **Redux Toolkit**: Complex apps with global state
- **Zustand**: Lightweight alternative to Redux
- **Jotai**: Atomic state management
- **Context API**: Built-in, good for simple state

### Data Fetching
- **TanStack Query**: Server state management, caching
- **SWR**: Alternative to React Query
- **Axios**: HTTP client with interceptors

### Form Handling
- **React Hook Form**: Performant forms with validation
- **Formik**: Popular alternative
- **Yup/Zod**: Schema validation

### Development Tools
- **Vite**: Fast build tool (recommended over CRA)
- **ESLint + Prettier**: Code quality and formatting
- **Husky + lint-staged**: Pre-commit hooks

## Vue Projects

### Essential Suggestions
- **Vue Router**: Official routing solution
- **Pinia**: Official state management (replaced Vuex)
- **TypeScript**: Optional but recommended for larger projects

### UI Libraries
- **Vuetify**: Material Design components
- **Element Plus**: Enterprise-focused UI
- **PrimeVue**: Rich component library

### Utilities
- **VueUse**: Collection of composition utilities
- **Vuelidate**: Form validation

## Laravel Projects

### Essential Suggestions
- **Laravel Sail**: Docker development environment
- **Laravel Sanctum**: SPA and API authentication
- **Laravel Horizon**: Queue monitoring (Redis required)

### Frontend Integration
- **Livewire**: Build reactive UI without JavaScript
- **Inertia.js**: Use React/Vue with Laravel routing
- **Laravel Mix/Vite**: Asset compilation

### Development & Testing
- **Laravel Telescope**: Debugging and monitoring
- **Pest PHP**: Modern testing framework
- **Laravel Debugbar**: Development debugging toolbar

### API Development
- **Laravel Passport**: Full OAuth2 implementation
- **API Resources**: Transform models to JSON
- **Laravel Scout**: Full-text search

## Node.js Backend Projects

### Framework Selection
- **Express**: Minimal and flexible
- **Fastify**: High performance alternative
- **NestJS**: Enterprise-grade with TypeScript
- **Koa**: From Express team, more modern

### Database & ORM
- **Prisma**: Modern ORM with great DX
- **TypeORM**: Decorator-based ORM
- **Mongoose**: MongoDB ODM

### Authentication
- **Passport.js**: Multiple auth strategies
- **jsonwebtoken**: JWT implementation
- **bcrypt**: Password hashing

### API Documentation
- **Swagger/OpenAPI**: API documentation
- **GraphQL**: Alternative to REST

### Development Tools
- **Nodemon**: Auto-restart on changes
- **dotenv**: Environment variables
- **Winston/Pino**: Logging
