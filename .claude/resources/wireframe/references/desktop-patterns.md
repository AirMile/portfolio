## Interaction Patterns

### Hover States

**All interactive elements need:**
```
Default:    [Button]        cursor: default
Hover:      [Button]  â†     cursor: pointer, visual change
Focus:      [Button]        visible outline (2px+)
Active:     [Button]        pressed state
Disabled:   [Button]        opacity: 0.5, cursor: not-allowed
```

**Hover Effects:**
- Background color change
- Underline (links)
- Shadow increase (cards)
- Scale (subtle, 1.02-1.05)
- Icon animation

```css
/* Visible focus for keyboard navigation */

/* Remove default on mouse, keep for keyboard */

  outline: none;
```
