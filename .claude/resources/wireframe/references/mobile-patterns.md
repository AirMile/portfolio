# Mobile Wireframe Patterns

Quick reference for mobile-first wireframe design. Use these guidelines when creating mobile wireframes.

---

## Touch Targets

### Minimum Sizes (WCAG 2.5.5 AAA)
- **Minimum touch target**: 44x44 CSS pixels
- **Recommended**: 48x48px for primary actions
- **Spacing between targets**: minimum 8px gap

### Touch Target CSS Pattern
```css
.touch-target {
  min-width: 44px;
  min-height: 44px;
  padding: 12px;
}

/* For inline targets (pagination, etc.) */
.inline-touch-target {
  display: inline-block;
  min-width: 44px;
  min-height: 44px;
  line-height: 44px;
  text-align: center;
}
```
